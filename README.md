#TwitCrusader

TwitCrusader is Twitter Client For Linux Desktop

## Are You A Developer?

Development Guidelines (Italian): http://dev.it.twitcrusader.org/

Developers Of TwitCrusader (Italian): http://it.twitcrusader.org/about/

##Required

* gcc or cc
* pkg-config
* intltool
* libcurl >= 7.22.0
* gtk+-2.4 >= 2.24.6
* oauth >= 0.9.1
* libxml-2.0 >= 2.6.32
* libnotify >= 0.7.3

### Compile - Makefile
    $ ./autogen.sh
    $ ./configure
    $ make
    $ su
    # make install

#### Unistalling - Makefile
    $ su
    # make uninstall
